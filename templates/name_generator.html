📋 Required Parts:
• Creator type: internal_ or freelancer-xxx_ or ramdam-xxx_ or influencer-xxx_
• Creative name with optional number
• All 4 tags: HOOK-xxx, VO-xxx, MUSIC-xxx, DIM (or empty "-")
• Feature tag (AIBG, STORY, LOGO, etc.)
• Language code (EN, FR, PT, etc. or lowercase)

🔍 Optional Parts:
• ID number (for AdManage format)
• Creator name (combined with type for non-internal)
• Date (DDMMYYYY format)
• Empty tags "-" (for compact format)

📌 Creative Naming Structure

Format: [id]_[creator]_[creator-type]_[filename]_[extra 4 tags]_[feature]_[language]_[date]
Critical Rules:
  • _ = separate main fields
  • - = separate details inside tags
  • ID, Creator, Date auto-added by Admanage
  • All 4 tags: HOOK → VO → MUSIC → DIM (always applied)
Examples:
internal_RedBottle_HOOK-come-text_VO-tom_MUSIC-rapbeat_PO_AIBG_en
internal_RedBottle_HOOK-any_VO-any_MUSIC-any_PO_AIBG_en
